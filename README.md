# 🎬 Netflix Data Visualization | Power BI Dashboard

This project focuses on exploring and visualizing Netflix data using **Power BI**.  
The goal was to clean, transform, and analyze the dataset to uncover insights about Netflix’s content — including movies, TV shows, genres, ratings, and production trends.

---

## 🔍 Project Overview
Netflix is one of the largest streaming platforms in the world, offering thousands of movies and TV shows.  
In this project, I built an **interactive Power BI dashboard** to understand patterns in content type, runtime, IMDb ratings, genres, and global distribution.

---

## 📊 Key Insights
- Total runtime analyzed: **1270+ hours**
- **562 Movies** and multiple TV Shows visualized
- **Average IMDb rating:** 6.03
- **Top Genres:** Comedy, Drama, Documentary
- **Content growth** from 2000 onwards
- **Subscribers by country** shown on an interactive world map

## 🚀 How to Use
1. Open the `.pbix` file in **Power BI Desktop**.  
2. Explore the visuals and apply filters by year, genre, or title.  
3. Gain insights from the interactive dashboard!
